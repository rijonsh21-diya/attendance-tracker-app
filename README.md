# Attendance Tracker App
This is a Python application that helps students track their subject attendance. 
It allows users to record attendance and calculate whether they meet the required attendance percentage for exams.
In my university students must maintain at least 75% attendance to write exams. 
This project helps students monitor their attendance and avoid shortages.

## Features
- Track subject attendance
- Calculate attendance percentage
- Simple and easy to use

## Technologies Used
- Python

## How to Run
1. Download the project
2. Open the folder
3. Run:

python attendance_tracker.py
